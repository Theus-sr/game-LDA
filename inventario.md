# Sistema de Inventário

## Objetivo
Documentar a estrutura inicial do sistema de inventário do jogo.

## Funcionalidades Planejadas
- Armazenar itens coletados pelo jogador.
- Exibir lista de itens no inventário.
- Permitir adicionar e remover itens.
- Controlar quantidade de cada item.

## Estrutura Proposta

Item:
- nome
- descrição
- quantidade

Inventário:
- adicionarItem()
- removerItem()
- listarItens()
